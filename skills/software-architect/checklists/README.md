# checklists/

One checklist file per phase (`00-calibration-checklist.md` … `17-review-checklist.md`, 18 total). Actionable, checkbox-style items derived 1:1 from the corresponding `quality-gates/` file — used for continuous self-assessment during a phase, distinct from the gate itself (which is the formal exit checkpoint).

Format standardized in `rules/checklist-structure.md`.
