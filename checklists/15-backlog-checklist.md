# Backlog Checklist

- [ ] Every `TASK-XXX` ID is unique and correctly formatted
- [ ] Every `TASK-XXX` has a non-empty `traces_to` pointing to an existing `US-XXX` or `UC-XXX`
- [ ] Every `US-XXX` in a currently-included milestone is covered by at least one `TASK-XXX`
- [ ] `definition_of_ready` is recorded once and applied consistently
- [ ] Non-story items trace to whatever they actually enable
- [ ] Priority reflects actual relative order within its milestone
- [ ] Every backlog item was explicitly confirmed by the user
