# SCR-002 — Create Task
*Traces to: UC-001 · Author: Priya*

**Description**: A form/modal for creating a task, reachable from the Task Board.

**Composition**: TaskForm (title, description, assignee fields) → SubmitButton

**Calls**: API-001

**Permission**: Any authenticated member of the tenant, scoped to their own tenant (tenant isolation per REQ-003/ARCH-002).

**States** (only Empty diverges from the index's General conventions)
| State | What the user sees |
|---|---|
| Empty | Not applicable — the form starts empty by definition |

**Responsive behavior**
Full-screen modal below the mobile breakpoint; centered dialog at tablet and above.

**Analytics events**
`task_create_submitted`, `task_create_succeeded`, `task_create_failed` (with the failure's `error.code`).
