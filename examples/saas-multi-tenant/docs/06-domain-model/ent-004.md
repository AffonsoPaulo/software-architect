# ENT-004 — User
*Kind: Entity · Traces to: UC-001*

| Attribute | Meaning |
|---|---|
| name | The person's display name. |
| email | Unique login/contact address within the tenant. |
| role | Determines permissions — e.g. Team Member vs. Project Admin. |
| tenant reference | The Tenant (ENT-001) this user account belongs to. |

**Invariants**
- A user belongs to exactly one tenant — no cross-tenant user accounts (a person on two customer teams needs two separate accounts).

**Belongs to aggregate**: itself is the root
