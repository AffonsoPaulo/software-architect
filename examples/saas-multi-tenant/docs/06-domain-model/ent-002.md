# ENT-002 — Project
*Kind: Entity · Traces to: UC-001*

| Attribute | Meaning |
|---|---|
| name | The project's display name within its tenant. |
| tenant reference | The Tenant (ENT-001) this project belongs to — fixes its data-isolation boundary. |

**Invariants**
- A project always belongs to exactly one tenant, set at creation and never changed.

**Belongs to aggregate**: itself is the root
