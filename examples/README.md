# examples/

End-to-end worked walkthroughs proving the Skill works in practice, not just in theory.

- `small-cli-tool/` — small project that skips several phases via Calibration (e.g. Frontend Planning), demonstrates scope calibration and Agile confirmation mode, and includes a second `cycle` (incremental feature added after `ready_for_implementation: true`) to demonstrate the reentry mechanism
- `saas-multi-tenant/` — larger project running all 18 phases, demonstrates Strict confirmation mode and real `[confirmação individual]` decisions

Each example includes its final `project-state.md`, every document produced, a `transcript.md` with real dialogue excerpts, and a clean run of `scripts/self-test.mjs`.

Populated in `plan-24-examples-walkthrough.md`.
